TO GET CONNECTION STRING (if needed):
═══════════════════════════════════════════════════════════

1. Click ⚙️ Settings (left sidebar)
2. Click "Database"
3. Click "Connection Pooling" tab
4. Select "Session mode"
5. Copy the connection string
6. Replace [YOUR-PASSWORD] with actual password

Format:
postgresql://postgres.[ID]:[PASSWORD]@aws-region.pooler.supabase.com:5432/postgres

COMMON QUESTIONS:
═══════════════════════════════════════════════════════════

Q: How do I log in to Supabase?
A: Use the same email/password you signed up with,
   or GitHub if you used GitHub login

Q: I don't see my project?
A: You might have multiple accounts, or project
   was deleted. Create a new one if needed.

Q: Where is the Resume button?
A: Top right corner, or a banner at the top of
   the page saying "This project is paused"

Q: How long does it take to resume?
A: Usually 30 seconds to 2 minutes. Be patient!

Q: What if it's already Active?
A: Then the problem is something else. Check your
   .env file has the correct DATABASE_URL

Q: Database keeps pausing?
A: Free tier pauses after 7 days inactivity.
   Either use it more, or upgrade to paid plan.
