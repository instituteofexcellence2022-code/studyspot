```

REACT_APP_PORTAL_TYPE=owner

REACT_APP_VERSION=1.0.0
NODE_ENV=production
```

### 2e) Deploy!
- Click "Deploy"
- Wait 3-5 minutes
- Copy your live URL (e.g., `https://om-owner.vercel.app`)
