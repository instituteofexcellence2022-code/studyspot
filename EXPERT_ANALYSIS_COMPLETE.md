**Location**: `web-owner/src/components/common/ProtectedRoute.tsx`

**Problem Identified**:

- Made real login/registration impossible

**Code Removed**:
```typescript

  if (!isAuthenticated && !isLoading) {
    const mockUser = {
      id: 'demo-user-123',

      lastName: 'Owner',

```

**Replaced With**:
